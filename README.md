This project is an Animal Shelter website designed to help users view available animals, learn their details, and apply for adoption through a simple and clean interface. The website is built with a modern layout and supports both English and Turkish language options.

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`
